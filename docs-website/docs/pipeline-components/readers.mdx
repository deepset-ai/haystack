# Readers

Readers are pipeline components that pinpoint answers in documents. They’re used in extractive question answering systems.

Currently, there's one Reader available for Haystack: [TransformersExtractiveReader](readers/transformersextractivereader.mdx), provided by the `transformers-haystack` integration package.
