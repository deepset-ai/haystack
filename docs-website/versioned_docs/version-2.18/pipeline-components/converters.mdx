# Converters

Use various Converters to extract data from files in different formats and cast it into the unified document format. There are several converters available for converting PDFs, images, DOCX files, and more.

| Converter                                                    | Description                                                                                                   |
| :----------------------------------------------------------- | :------------------------------------------------------------------------------------------------------------ |
| [AzureOCRDocumentConverter](/docs/azureocrdocumentconverter) | Converts PDF (both searchable and image-only), JPEG, PNG, BMP, TIFF, DOCX, XLSX, PPTX, and HTML to documents. |
| [CSVToDocument](doc:csvtodocument)                           | Converts CSV files to documents.                                                                              |
| [DocumentToImageContent](doc:documenttoimagecontent)         | Extracts visual data from image or PDF file-based documents and converts them into `ImageContent` objects.    |
| [DOCXToDocument](doc:docxtodocument)                         | Convert DOCX files to documents.                                                                              |
| [HTMLToDocument](/docs/htmltodocument)                       | Converts HTML files to documents.                                                                             |
| [ImageFileToDocument](doc:imagefiletodocument)               | Converts image file references into empty `Document` objects with associated metadata.                        |
| [ImageFileToImageContent](doc:imagefiletoimagecontent)       | Reads local image files and converts them into `ImageContent` objects.                                        |
| [JSONConverter](doc:jsonconverter)                           | Converts JSON files to text documents.                                                                        |
| [MarkdownToDocument](/docs/markdowntodocument)               | Converts markdown files to documents.                                                                         |
| [MSGToDocument](doc:msgtodocument)                           | Converts Microsoft Outlook .msg files to documents.                                                           |
| [MultiFileConverter](doc:multifileconverter)                 | Converts CSV, DOCX, HTML, JSON, MD, PPTX, PDF, TXT, and XSLX files to documents.                              |
| [OpenAPIServiceToFunctions](/docs/openapiservicetofunctions) | Transforms OpenAPI service specifications into a format compatible with OpenAI's function calling mechanism.  |
| [OutputAdapter](/docs/outputadapter)                         | Helps the output of one component fit into the input of another.                                              |
| [PDFMinerToDocument](/docs/pdfminertodocument)               | Converts complex PDF files to documents using pdfminer arguments.                                             |
| [PDFToImageContent](doc:pdftoimagecontent)                   | Reads local PDF files and converts them into `ImageContent` objects.                                          |
| [PPTXToDocument](/docs/pptxtodocument)                       | Converts PPTX files to documents.                                                                             |
| [PyPDFToDocument](/docs/pypdftodocument)                     | Converts PDF files to documents.                                                                              |
| [TikaDocumentConverter](/docs/tikadocumentconverter)         | Converts various file types to documents using Apache Tika.                                                   |
| [TextFileToDocument](/docs/textfiletodocument)               | Converts text files to documents.                                                                             |
| [UnstructuredFileConverter](/docs/unstructuredfileconverter) | Converts text files and directories to a document.                                                            |
| [XLSXToDocument](doc:xlsxtodocument)                         | Converts Excel files into documents.                                                                          |
