# Fetchers

Currently, there's one Fetcher in Haystack: LinkContentFetcher. It fetches the contents of the URLs you give it.

| Component                                    | Description                                                                                  |
| :------------------------------------------- | :------------------------------------------------------------------------------------------- |
| [LinkContentFetcher](doc:linkcontentfetcher) | Fetches the contents of the URLs you give it so you can use them as data for your pipelines. |
